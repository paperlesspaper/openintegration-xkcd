Implementation of Open Integration: https://github.com/paperlesspaper/paperlesspaper-apps/blob/main/OPENINTEGRATION.md

Create an open integration displaying the daily xkcd on a website.
It should complety fill out (contain) the screen with 2em padding.

Add the following options as params in the url
?kind=random
?difference=356 // displays XKCD from one year ago

Use static html/js and host it on Vercel.
