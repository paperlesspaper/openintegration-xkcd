# OpenPaper Integration Sample

This is a sample for a very simple implementation using the OpenIntegration of [paperlesspaper](https://paperlesspaper.de).

## Overview

This project renders a minimal daily XKCD view that can be used in the [paperlesspaper project](https://github.com/paperlesspaper).

## How to use

Inside the app select `New Image` > `Custom Integration` and enter the url:
`https://openintegration-dailyxkcd-gamma.vercel.app/config.json`

## Key files

- [render.html](render.html): Static rendering template used for the display.
- [api/xkcd.js](api/xkcd.js): Serverless endpoint that fetches XKCD data and returns it in a lightweight format.
- [config.json](config.json): Simple runtime configuration used by the Open Integrations.
- [vercel.json](vercel.json): Vercel routing and build configuration.

## API behavior

The `api/xkcd.js` endpoint requests the latest XKCD JSON feed, extracts the minimal fields needed for the display, and returns them for consumption by the `render.html`

## Deployment

Deploy to Vercel and ensure the `api` directory is recognized as serverless functions. The static rendering uses `render.html` and does not require a build step.
